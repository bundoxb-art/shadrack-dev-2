How I Solve Real Problems

Building PesaVoucher — A Complete FinTech Ecosystem
Architecting a four-platform payment voucher system with full MPesa integration, solo.
FINTECH::FULL-STACK
PROBLEM
Client needed a complete payment voucher system — user app, merchant portal, admin back-office, and MPesa — from a single developer.
SOLUTION
Built with Node.js + Spring Boot microservices, Vue.js frontends, shared TypeScript SDK. Three portals + mobile app on one backend.
RESULT
4 production platforms live — app.pesavoucher.com, admin.pesavoucher.com, merchant.pesavoucher.com, and Play Store mobile app.
NHIF & NSSF — Government Apps Serving Millions
Enterprise architecture that made critical government apps fast, secure, and reliable at scale.
GOV::ENTERPRISE
PROBLEM
Kenya's NHIF and NSSF apps needed to handle millions of concurrent users with secure auth, offline capabilities, and real-time sync.
SOLUTION
Implemented MVVM with Clean Architecture in Flutter and Kotlin. Smart caching, optimized API pagination, biometric auth, encrypted local storage.
RESULT
App load time reduced 40%. Crash rate below 0.1%. Both apps serve millions of active Kenyan citizens with consistent 4.5+ star ratings.
